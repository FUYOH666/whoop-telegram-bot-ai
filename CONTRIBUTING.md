# Contributing to RAS Bot

Спасибо за интерес к проекту! Мы рады любым улучшениям и идеям.

Thank you for your interest in the project! We welcome any improvements and ideas.

## Как внести вклад / How to Contribute

### Идеи и предложения / Ideas and Suggestions

Мы открыты к любым идеям! Вот что можно улучшить:

We're open to any ideas! Here's what can be improved:

- **Дополнительные интеграции** / **Additional Integrations**
  - Apple Health, Google Fit, Garmin, Fitbit
  - Календари (Google Calendar, Apple Calendar)
  - Task managers (Todoist, Notion, etc.)

- **Улучшения функциональности** / **Feature Improvements**
  - Поддержка нескольких пользователей / Multi-user support
  - Веб-дашборд для статистики / Web dashboard for statistics
  - Экспорт данных / Data export
  - Больше настроек промптов / More prompt customization options

- **Улучшения UX** / **UX Improvements**
  - Более гибкая настройка времени слотов / More flexible slot time configuration
  - Напоминания о пропущенных слотах / Reminders for missed slots
  - Групповые статистики / Group statistics

- **Технические улучшения** / **Technical Improvements**
  - Улучшение производительности / Performance improvements
  - Добавление тестов / Adding tests
  - Улучшение документации / Documentation improvements
  - Переводы на другие языки / Translations to other languages

### Процесс / Process

1. **Откройте Issue** / **Open an Issue**
   - Опишите вашу идею или проблему / Describe your idea or problem
   - Используйте русский или английский язык / Use Russian or English
   - Добавьте метки, если возможно / Add labels if possible

2. **Обсуждение** / **Discussion**
   - Мы обсудим идею и определим, как лучше её реализовать
   - We'll discuss the idea and determine the best way to implement it

3. **Pull Request** (опционально) / **Pull Request** (optional)
   - Если вы хотите реализовать идею сами, создайте PR
   - If you want to implement the idea yourself, create a PR
   - Следуйте стилю кода проекта / Follow the project's code style
   - Добавьте тесты, если возможно / Add tests if possible

### Стиль кода / Code Style

- Используем `ruff` для форматирования и линтинга / We use `ruff` for formatting and linting
- Типизация через `pyright` / Type checking via `pyright`
- Документация на русском и английском / Documentation in Russian and English

### Вопросы? / Questions?

Не стесняйтесь открывать Issue для любых вопросов!

Don't hesitate to open an Issue for any questions!

---

**Спасибо за ваш вклад!** / **Thank you for your contribution!**
